8.2 Tutorial

Change the API configuration format of the IP address：{base_url}/sv5-server/refresh_proxy_chain/{user_id}/{port}

For example, the public IP address of the server is 123.45.67.89, and the account ID is 26

Then {base_url} is 123.45.67.89:12569

{user_id} is 26

{port} is the port created in the background

If I want to refresh the IP of port 24000 via the API, configure it as:

http://123.45.67.89:12569/sv5-server/refresh_proxy_chain/26/24000

Open the above link in your browser to complete the IP switch

Change the country API configuration format：{base_url}/sv5-server/update_area/{user_id}/{port}/{area}

For example, the public IP address of the server is 123.45.67.89, and the account ID is 26

Then {base_url} is 123.45.67.89:12569

{user_id} is 26

{port} is the port created in the background

{area} is the country you want to change. For example, the United States is us

If I want to change the country of 24000 port to us via API, the configuration is:

http://123.45.67.89:12569/sv5-server/update_area/26/24000/us

To complete the country switch, open the above link in your browser

To locate a continent or city, add the name of the continent and city after the country, separated by a -

For example:

Locate to the continent: us-mississippi

Locate to the city: us-mississippi-Philadelphia

No.5 Frequently Asked Questions

1. After the port is configured, the port is unavailable and the IP address cannot be obtained

1.1: Check whether the account and password are incorrectly configured. The format of the account and password is user:password, separated by semicolons

1.2: After the port configuration is complete, the connection time takes 3s-10s. After refreshing the port, wait a few seconds before using the port configuration

1.3: You can refresh the port to obtain the ip address again

1.4: You can try to delete this port and rebuild it

2. The port will be open, the target page cannot be opened (add prohibited domain name list)

2.1: The port is occupied by the system or another application

2.2: Some domain names and addresses have been blacklisted, please consult the business manager for details

3. Reasons for inaccurate traffic statistics (whitelisting and password necessity)

Configure account passwords for all ports and update them periodically to prevent traffic quota loss caused by account configuration leakage. You can also configure a whitelist to ensure traffic security on the current device

4. The country and region configured for the port do not match the example

Refresh the port on ZennoProxy to obtain the IP address again

5. The dynamic proxy IP duration is too short

Check whether the background configuration option StickyTime is set to 15 minutes or 30 minutes. If not, the sticktime is set to 5 minutes by default

6. How to use Zone and Bypasslist? What does it do?

Add the Zone and Bypasslist in the ZennoProxy background Settings. Select the Zone and Bypasslist when adding ports.

Zone: Set a group for multiple users or multiple ports

Bypasslist: Add a URL by bypasslist port. The URL bypasses the proxy connection, that is, the local connection is directly used.
